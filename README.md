## Run Locally

Clone the project

```bash
  git clone https://github.com/firstChairCoder/Medvice_Test.git
```

Go to the project directory

```bash
  cd Medvice_Test
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
